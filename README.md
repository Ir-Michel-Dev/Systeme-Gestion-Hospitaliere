# Systeme-Gestion-Hospitaliere
Le Système de Gestion Hospitalière développé par DigitalSoft Congo est une solution informatique complète conçue pour moderniser et simplifier la gestion des centres de santé, cliniques et hôpitaux.  Cette application permet une gestion centralisée des patients, des consultations, de la facturation, des paiements et des rendez-vous.
